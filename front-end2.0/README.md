# Instituto Tecnológico Sitio Uno

Landing page institucional construida con React y Vite, refactorizada desde el front-end estático original.

## Estructura

- `src/components`: header, hero, valores y footer reutilizables.
- `src/data`: contenido centralizado para la página.
- `src/hooks`: lógica de comportamiento del header al hacer scroll.
- `public/ITSU.webp`: logo institucional.

## Scripts

- `npm run dev`: desarrollo local.
- `npm run build`: compilación de producción.
- `npm run lint`: validación de estilo.

## Notas

La interfaz conserva la identidad visual del sitio anterior, pero ahora está organizada de forma modular para facilitar mantenimiento y crecimiento.
